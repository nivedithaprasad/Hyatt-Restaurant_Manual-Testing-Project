# Manual Testing Project – Hyatt Restaurant

## Project Overview

This project demonstrates end-to-end manual testing activities performed for the Hyatt Restaurant application using Agile methodology and Jira.

## Tools Used

* Jira
* Zephyr Scale
* Microsoft Excel
* Google Chrome

## Testing Activities Performed

### Requirement Analysis

* Analyzed the project requirements and identified test scenarios for the Hyatt Restaurant application.

### Agile Planning

* Created Epic in Jira.
* Created User Stories under the Epic.
* Created Sub-Tasks for testing activities.

### Test Case Design

* Designed 30 test cases covering functional and negative scenarios.
* Maintained the test cases in the uploaded Excel sheet and in Zephyr.

### Test Execution

* Executed all test cases using Zephyr.
* Recorded Pass and Fail results.
* Updated execution status in Jira.

### Bug Reporting

* Logged bugs for failed test cases.
* Captured:

  
- Summary
- Steps to Reproduce
- Test Data
- Expected Result
- Actual Result
- Severity
- Priority
- Workaround
- Screenshots
- Status

## Deliverables

* Epic in Jira
* User Stories in Jira
* Sub-Tasks in Jira
* Test Cases in Excel and Zephyr
* Test Execution Results
* Bug Reports
* Jira Screenshots

## Project Flow

Epic → User Story → Sub-Tasks → Test Case Design → Test Case Execution → Bug Reporting → Bug Tracking → Closure
