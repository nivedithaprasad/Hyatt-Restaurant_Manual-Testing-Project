# Test Cases

This folder contains manual test cases created for the Hyatt Restaurant web application.

The test cases were designed based on user stories and cover multiple modules including:

* Login Module
* Home Page Module
* Product Module

Each test case contains:

* User Story ID
* Test Scenario
* Test Cases
* Preconditions
* Test Steps
* Test Data
* Expected Result
* Actual Result
* Status (Pass/Fail)

Additional details included in the test case document:

* Tester Name
* Creation Date
* Review Information

Tool Used:

* Microsoft Excel

Total Test Cases: 30
