# Jira Artifacts

This folder contains Jira and Zephyr artifacts created for each module of  the Hyatt Restaurant Manual Testing Project.

Project Workflow

1. Created Epics for major application modules.
2. Created User Stories under each Epic.
3. Added Summary, Definition of Done (DoD), and Acceptance Criteria.
4. Created Subtasks for individual testing activities.
5. Designed test cases using Zephyr.
6. Executed test cases and marked them as Pass or Fail.

Artifacts Included

* Epic Details
* User Stories
* Definition of Done (DoD)
* Acceptance Criteria
* Subtasks
* Zephyr Test Cases
* Test Execution Results

Tools Used

* Jira
* Zephyr

Modules Covered

* Login Module
* Home Page Module
* Product Module
