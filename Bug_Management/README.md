
# Bug Management

This folder contains bug tickets identified during manual testing of the  Hyatt Restaurant Application.

## Defect Tracking Process
1. Execute test cases.
2. Identify deviations from expected behavior.
3. Log defects in Jira with complete details.
4. Assign severity and priority.
5. Attach screenshots and supporting evidence.
6. Track defect status until resolution.

## Information Included in Each Bug Ticket
- Bug Summary
- Steps to Reproduce
- Test Data
- Expected Result
- Actual Result
- Severity
- Priority
- Workaround
- Screenshot

## Tools Used
- Jira
- Zephr Essential
- Google Sheets
- Chrome Browser

Manual Testing
